Recycle your features

SmartBlocks allows you to build web apps by assembling features blocks.

Why SmartBlocks

As developers, when building web apps, we often find ourselves making the same basic features over and over.

SmartBlocks gives you the possibility to start building your real features without starting over every time, reusing features you wrote instead of rewriting them.

Just drop'em

To add blocks to your web app, you just need to drop its folder into the framework's plugins folder. And that's it. Then you can combine their features with your own blocks, and thus create a complex DNA for your web app.

Need to create a todo list system, but you want to be able to plan your tasks ? Just add the Time Block, which contains a calendar and events, that you can easily use within your tasks system. Need to add notifications when you're supposed to start a task ? Just add the Notifications block.

Live and local

SmartBlocks lets you create a single page web app, giving you tools to make the most of it.

Your app needs to respond really fast ? No problem. Data synchronization is handled in the background.

Your app's users need to collaborate live ? No problem either. SmartBlocks gives you a way to use websockets.
