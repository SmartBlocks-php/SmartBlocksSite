SmartBlocks

Build complete web apps.
Not complicated web apps.

SmartBlocks is a web development framework that helps structuring your front-end, dividing it into reusable modules.

SmartBlocks brings structure to your front-end, giving you a complete execution environment in the browser.

SmartBlocks makes your features reusable and easily sharable with the community.

SmartBlocks is delivered with tools to create a RESTful back-end, so that you can persist your data and use it for other interfaces (like mobile applications).

The web is evolving

Today we're passing from the traditional server app with generated with which a user dialogs through forms to full front-end applications, in which the server becomes a data persistence structure.

Websites used to be sequences of webpages loading in the browser.

Today we're starting to have full featured single-page web applications.

This evolution brings many advantages

Single-page applications offer an experience that is very similar to desktop applications.

Screen transitions can be very fast and smooth. The application is way more reactive, and everything can be connected.

Data persistence is well separated from the interface, and if well made can be reused in other types of interfaces, like mobile applications.

But it can also bring problems

As the app becomes more and more complex, it usually loses code simplicity.

This lack of structure can also bring problems when the developer wants to reuse features. We find ourselves making copy/paste of small code portions, which is kind of messy.

As the app gains features, the front-end code can quickly become unstructured and difficult to maintain.

That's where SmartBlocks comes in

Structure for your apps

Everything's divided into feature blocks. These blocks are put into a main project structure, which brings a back-end environment and a front-end environment.
SmartBlocks is based on libraries like Backbone and Require.js.

Better maintainability

When you want to add features to your web application, just create a new block with its own data, views and algorithms.

Reuse and share

Once you've created your blocks, and worked with them, they can be very easily reused in other SmartBlocks projects. You will also be able to share them with the SmartBlocks community by releasing them in a Block Store.

Where do I go from here ?

If you want to try out all this, go on to the Getting started tutorials

You can also browse feature blocks that have already been shared in the BlockStore

SmartBlocks is a currently an open-source student project. You can start playing with it, but don't use it in production as it is still very unstable and subject to many future changes.

If the concept appeals to you and if you want to join our (very small) developper community, fork the project on GitHub.
